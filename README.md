# FilmLightMeter

Экспонометр для плёночной фотографии под Android. Использует камеру телефона для замера яркости сцены и рассчитывает экспопару (выдержку и диафрагму) с учётом выбранной плёнки.

## Возможности

- **Замер по камере** с тремя режимами: точечный, центровзвешенный, матричный
- **Расчёт экспопары** по системе APEX: EV = log₂(N²/t), EV₁₀₀ = log₂(L·S/K)
- **Приоритет диафрагмы** или **приоритет выдержки** — фиксируете одно, приложение считает второе
- **30+ пресетов популярных плёнок**: Portra 160/400/800, Ektar 100, Gold 200, Tri-X 400, HP5+, T-Max, Delta, Velvia 50/100, Provia, Ektachrome, Cinestill 400D/800T и другие
- **Компенсация экспозиции ±EV** (шаг 1/3 стопа) для push/pull
- **Поддержка ND-фильтров**
- **Коррекция закона взаимности** (reciprocity failure) для длинных выдержек
- **Калибровка** — точная подстройка под конкретное устройство
- **Ретро-дизайн** со стрелочным циферблатом в стиле Sekonic L-398

## Как собрать APK

### Вариант 1: GitHub Actions (автоматически)

Каждый push в `main` запускает сборку. Готовые APK (debug и release) доступны в разделе **Actions → последний запуск → Artifacts**.

Если создать тег `v1.0.0` — APK автоматически прикрепятся к GitHub Release.

### Вариант 2: Локально

```bash
git clone <repo>
cd FilmLightMeter
./gradlew :app:assembleDebug
# APK: app/build/outputs/apk/debug/app-debug.apk
```

Требования: JDK 17+, Android SDK 34, Gradle 8.7 (подтянется автоматически через wrapper).

## Как установить

1. Скачайте `app-debug.apk` или `app-release.apk` из Releases / Actions Artifacts
2. Разрешите установку из неизвестных источников в настройках Android
3. Установите APK
4. При первом запуске разрешите доступ к камере

## Как пользоваться

1. **Выберите плёнку** — нажмите «Выбрать» в карточке плёнки
2. **Выберите режим замера** — точка/центр/матрица
3. **Направьте камеру** на сцену — стрелка EV оживёт
4. **Нажмите «Lock»** 🔒 в правом нижнем углу превью, чтобы зафиксировать замер
5. **Выберите приоритет**: диафрагма или выдержка — нажмите «приоритет» под нужной колонкой
6. **Подстройте** диафрагму или выдержку — приложение покажет вторую величину

Для длинных выдержек (> 1 сек) включена коррекция закона взаимности — плёнка теряет чувствительность и это учитывается. Можно выключить в настройках (⚙).

### Как считается EV

Телефон в автоэкспозиции выставляет свои `N, t, S` (диафрагма, выдержка, ISO). По `SENSOR_EXPOSURE_TIME`, `SENSOR_SENSITIVITY`, `LENS_APERTURE` из CaptureResult мы знаем, с какими параметрами телефон снял кадр. Из Y-канала YUV вычисляем среднюю яркость кадра. Если она отличается от среднего серого (≈118 из 255), добавляем поправку log₂(Y/128):

```
EV_cam = log2(N² / t)
EV100 = EV_cam - log2(S / 100) + log2(avgY / midGray)
```

Затем EV₁₀₀ пересчитывается под ISO выбранной плёнки и раскладывается в требуемую экспопару:

```
t = N² / 2^EV_iso        — выдержка
N = √(t · 2^EV_iso)      — диафрагма
```

### Калибровка

Формула даёт результат в пределах 0.5–1 стопа от референса для большинства устройств. Если приложение «переэкспонирует» или «недоэкспонирует» — подстройте слайдер «Калибровка» в настройках (⚙) на требуемое число стопов.
